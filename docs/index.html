SUKESUKE-Pump-Maki

ダウンロード  Github

インストール

ダウンロードからSUKESUKE-Pump-Maki-master.zipをダウンロードし、中のPluginフォルダをAviutlのインストールされたフォルダにコピーしてください。

使い方

まず、ゲームの映像にアニメーション効果から「ｻﾝｼｬｲﾝ画面(設定から)」を選択します。 画面上に青と赤の点が表示されるので、それぞれがタンク・HP表示に収まるように設定の値を変更して調整します。 同様にマキマキに「ﾀﾝｸ」、ゆかりさんに「HP」を貼り付けると、それぞれタンク残量とHPに応じて衣装が変化します。
